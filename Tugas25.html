Mengenal semantic elements di HTML5

  • Article

    Ketika kita ingin menggunakan layout untuk menampilkan artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer. Karena tag article memiliki tipe block. artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya

  • Aside

    Element ini kita gunakan untuk melengkapi layout utama. Bila kita memiliki blog maka kita bisa menabahkan element aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.Karena aside memiliki sifat block maka lebernya sesuai browser bila kita tidak menentukannya
